# 🏘️ communityos

A New Flutter project.

## Getting Started

CommunityOS is a modern Flutter application designed to simplify apartment and gated community management. It provides residents with a single platform to manage daily activities, book amenities, receive society notices, manage vehicles, and access their personal profile.

The application focuses on providing a clean UI, smooth user experience, and scalable architecture that can be integrated with backend services for real-world deployments.

---

## ✨ Features

### 👤 Resident Profile
- View and edit resident profile
- Update personal information
- Upload profile picture
- Emergency contact management
- Secure logout

### 🚗 Vehicle Management
- Add multiple vehicles
- Select vehicle type
- Store vehicle registration numbers
- Manage resident vehicles

### 🏊 Amenity Booking
Book society amenities including:

- Gym
- Swimming Pool
- Badminton Court
- Club House
- Garden Lawn
- Kids Play Area

Features:
- View operating hours
- Book available slots
- Booking confirmation
- Booking details page

### 📢 Society Notices
Stay updated with:

- Maintenance announcements
- Water supply notices
- Society meetings
- Festival celebrations
- Safety notifications

### 📊 Activity Dashboard
Interactive analytics including:

- Today's activities
- Amenity usage statistics
- Peak usage hours
- Beautiful charts using FL Chart

### 📱 Modern UI
- Material Design 3
- Light & Dark Theme
- Responsive layouts
- Smooth page transitions
- Clean and minimal design
